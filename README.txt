IP UDP raw сокет
client формирует заголовки udp, ip и eth и отправляет сообщение, после ожидает ответа
server работает с raw сокетом, как с обычным udp
серверу необходимо сообщить его IP адрес
необходимо запускать на двух разных компьютерах, клиенту необходимо сообщить имя используемого интерфейса, MAC адрес сервера, IP адреса
